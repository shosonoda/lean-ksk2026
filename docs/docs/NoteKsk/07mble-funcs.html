    Chapter 07: Measurable functions #

    This file follows blueprint/src/chapters/07mble-funcs.tex.

    Mathlib already provides the central notions:

    • measurable spaces are MeasurableSpaces;
    • measurable maps and measurable functions are Measurable;
    • the Borel structure on EReal is the standard order-topology Borel space;
    • almost-everywhere statements are expressed by the filter Measure.ae.

    The lecture-note definitions are therefore mostly recorded as aliases in Defs.lean, and this chapter supplies the corresponding theorem names.

    Two small changes from the notes are intentional and documented near the statements:

    • for the main measurable-function criterion, the fully proved Lean statement uses all EReal thresholds. The lecture version using only real thresholds is also recorded as a named bridge theorem;
    • real reciprocal is available in mathlib as the total function with 0⁻¹ = 0. We still record the note's nonzero-domain version.
