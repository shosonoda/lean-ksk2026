    Chapter 03: Lebesgue outer measure #

    This file follows blueprint/src/chapters/03lebesgue-outer.tex.

    The lecture note first defines Lebesgue outer measure by countable coverings with left half-open rectangles. In mathlib, Lebesgue measure is already available as volume; applying a measure to an arbitrary set gives its outer-measure value. We therefore use

    • Fin d → ℝ as the model of ℝ^d;
    • (volume : Measure (Fin d → ℝ)) as Lebesgue outer measure on all subsets;
    • explicit boxes and cover-cost definitions to keep the blueprint definition visible.

    Although the lecture notes place Jordan measure before Lebesgue outer measure, this file does not import Chapter 02. The few Jordan compatibility statements below use only the shared definitions from Defs.lean and mathlib's volume.
