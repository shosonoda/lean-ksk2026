    Chapter 01: Riemann integration #

    This file follows blueprint/src/chapters/01riemann.tex.

    It records the definition layer for one-dimensional Riemann and Darboux integration. Deferred chapter statements whose proofs are still placeholders live in NoteKsk/01riemann-statements.lean, so downstream measure-theory files can import this module without inheriting proof placeholders.
