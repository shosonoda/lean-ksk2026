    Compatibility with mathlib volume #

    source
    theorem NoteKsk.Chapter02.jordanMeasure_eq_volume_of_jordanMeasurable {d : ℕ} {A : Set (Space d)} (hA : JordanMeasurable A) :
    jordanMeasure A = MeasureTheory.volume A

    On Jordan-measurable sets, Jordan measure agrees with the value of mathlib's Lebesgue measure. Chapter 03 uses this to compare with λ*.

    source
    theorem NoteKsk.Chapter02.volume_iUnion_eq_tsum_of_jordanMeasurable {d : ℕ} (A : ℕ → Set (Space d)) (hdisj : ∀ ⦃m n : ℕ⦄, m ≠ n → Disjoint (A m) (A n)) (hA : ∀ (n : ℕ), JordanMeasurable (A n)) :
    MeasureTheory.volume (⋃ (n : ℕ), A n) = ∑' (n : ℕ), MeasureTheory.volume (A n)

    For pairwise disjoint Jordan-measurable sets, the outer value of volume is countably additive. This is stated here as the bridge needed in Chapter 03.
