    Chapter 05: Lebesgue measurable sets and Lebesgue measure #

    This file follows the active blueprint file blueprint/src/chapters/05lebesgue03.tex.

    We use mathlib's completed measurable-set predicate NullMeasurableSet for Lebesgue measurable sets. The chapter-specific inner/outer measure criteria are recorded as derived theorems, so later files can use the lecture-note names while the closure and additivity results are proved by the existing measure API.
